Draw obstacles with the first tool; erase with the second. Use velocity to push fluid (drag); use particles to add tracers. Velocity on the four sides can be set via Dirichlet conditions in the text boxes (Ux, Uy) or via Neumann-type boundary conditions as zero gradient using the “Zero ∇” checkboxes.
